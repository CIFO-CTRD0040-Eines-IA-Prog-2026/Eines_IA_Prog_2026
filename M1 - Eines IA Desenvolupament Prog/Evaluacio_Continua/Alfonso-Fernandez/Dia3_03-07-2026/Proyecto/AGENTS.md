# AGENTS.md — Proyecto

## Ámbito
Este AGENTS.md se aplica exclusivamente a la carpeta `Proyecto` y sus subdirectorios. Cuando trabajes aquí, estas son tus reglas de oro.

## Reglas

1. **Stack permitido:** HTML, CSS y JavaScript. Si fuera necesario algún framework, librería externa, CDNs o preprocesador, añádelo.
2. **Idioma:** Contenido en castellano. Nombres de clases y ficheros en inglés o castellano, siempre en minúscula y con guión.
3. **Código:** Bien formateado (identación consistente, sin líneas muy largas, sin espacios innecesarios).
4. **Estilo visual:** Paleta oscuro #0B2B3C, teal #0E7C86, fondo claro. Sobrio y legible.
5. **Sin sorpresas:** Antes de hacer un cambio grande, proponme un plan y espera mi validación.
6. **Consulta solamente:** Si te pregunto algo sobre el código existente, responde sin hacer ninguna acción ni modificación.
7. **Verificación:** Después de cualquier cambio, verifica que el HTML, CSS y HTML sean válidos (estructura correcta, sin etiquetas sin cerrar, etc.).
8. **Respuestas:** Sé conciso. No añadas explicaciones de código a menos que te las pida explícitamente.
