Variables

Una variable és un espai a la memòria que guarda un valor. Cada variable té un tipus de dada que determina què pot contenir.

Tipus de dades

  • int — Nombres enters (ex: 5, -3)
  • float / double — Decimals (ex: 3.14)
  • char — Un sol caràcter (ex: 'A')
  • string — Text (ex: "Hola")
  • boolean — Cert o fals (true / false)

Declaració i assignació

Primer es declara el tipus i el nom, després s'hi assigna un valor.

int edat;
edat = 20;

float preu = 19.99;
string nom = "Anna";

Àmbit (scope)

Les variables poden ser globals (fora de funcions) o locals (dins d'un bloc).

// Variable global
int comptador = 0;

void exemple() {
  // Variable local
  int temporal = 5;
}

Condicionals

Les estructures condicionals permeten executar codi diferent segons si una condició és certa o falsa.

if / else

int nota = 7;

if (nota >= 5) {
  Console.WriteLine("Aprovat");
} else {
  Console.WriteLine("Suspès");
}

Si la condició és certa s'executa el primer bloc; si no, el segon.

else if (anidats)

int hora = 14;

if (hora < 12) {
  Console.WriteLine("Bon dia");
} else if (hora < 18) {
  Console.WriteLine("Bona tarda");
} else {
  Console.WriteLine("Bona nit");
}

switch

string color = "verd";

switch (color) {
  case "vermell":
    Console.WriteLine("Atura't");
    break;
  case "verd":
    Console.WriteLine("Endavant");
    break;
  default:
    Console.WriteLine("Color no reconegut");
    break;
}

Operadors de comparació i lògics

Comparació

  • == igual que
  • != diferent de
  • < / > menor / major que
  • <= / >= menor o igual / major o igual

Lògics

  • && AND — les dues condicions han de ser certes
  • || OR — almenys una condició ha de ser certa
  • ! NOT — inverteix el valor lògic
int edat = 18;
bool teCarnet = true;

if (edat >= 18 && teCarnet) {
  Console.WriteLine("Pots conduir");
}

Bucles

Els bucles repeteixen un bloc de codi mentre es compleixi una condició. N'hi ha de tres tipus principals.

for

S'usa quan sabem quantes vegades volem repetir.

// Del 0 al 4
for (int i = 0; i < 5; i++) {
  Console.WriteLine(i);
}

while

Repeteix mentre la condició sigui certa. Pot executar-se zero vegades.

int i = 0;
while (i < 5) {
  Console.WriteLine(i);
  i++;
}

do-while

Similar a while, però s'executa almenys una vegada.

int i = 0;
do {
  Console.WriteLine(i);
  i++;
} while (i < 5);

Comparativa: for vs while vs do-while

Bucle Quan s'usa Execució mínima Exemple típic
for Nombre conegut d'iteracions 0 Recórrer un array
while Condició variable, pot no entrar 0 Llegir fins a final de fitxer
do-while Condició variable, cal entrar almenys un cop 1 Menú d'opcions

Funcions

Una funció és un bloc de codi reutilitzable que fa una tasca concreta. Pot rebre dades (paràmetres) i tornar un resultat.

Definició bàsica

// Funció que no rep res i no torna res
void saludar() {
  Console.WriteLine("Hola!");
}

// Crida
saludar();

Paràmetres

void saludarPerNom(string nom) {
  Console.WriteLine("Hola, " + nom);
}

// Crida amb argument
saludarPerNom("Maria");

Valor de retorn

int suma(int a, int b) {
  return a + b;
}

int resultat = suma(3, 5);
Console.WriteLine(resultat); // 8

Funcions amb més d'un paràmetre

float mitjana(float a, float b, float c) {
  return (a + b + c) / 3;
}

Console.WriteLine(mitjana(7, 8, 9)); // 8

Funcions que en criden d'altres

int doblar(int x) {
  return x * 2;
}

int quadruplicar(int y) {
  return doblar(doblar(y));
}

Console.WriteLine(quadruplicar(3)); // 12

Variables locals i globals

int comptadorGlobal = 0;

void incrementa() {
  comptadorGlobal++;
}

Console.WriteLine(comptadorGlobal); // 0
incrementa();
Console.WriteLine(comptadorGlobal); // 1
