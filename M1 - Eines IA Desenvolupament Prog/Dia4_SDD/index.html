Programació

La programació és el procés d'escriure instruccions que un ordinador pot executar. Un programa és una seqüència d'ordres que resol un problema concret.

Algorisme: conjunt de passos ordenats per resoldre un problema.

Codi font: text escrit en un llenguatge de programació.

Compilador / Intèrpret: tradueix el codi a instruccions de màquina.

// Hola, món!
console.log("Hola, món!");

Variables

Una variable és un contenidor amb nom on es guarda un valor. En JavaScript es declaren amb let, const o var.

let: valor que pot canviar.

const: valor constant (no es pot reassignar).

Tipus: number, string, boolean, undefined, null, object.

let nom = "Anna";
const edat = 20;
let esEstudiant = true;
console.log(nom, edat, esEstudiant);

Condicionals

Les estructures condicionals permeten executar diferents blocs de codi segons si una condició és certa o falsa.

if / else: executa un bloc si la condició és certa, un altre si és falsa.

else if: encadenar múltiples condicions.

switch: alternativa per a múltiples valors concrets.

let nota = 7;
if (nota >= 5) {
  console.log("Aprovat");
} else {
  console.log("Suspès");
}

Bucles

Els bucles repeteixen un bloc de codi mentre es compleixi una condició o per a cada element d'una col·lecció.

for: repeteix un nombre conegut de vegades.

while: repeteix mentre la condició sigui certa.

for…of: itera sobre elements d'un array.

for (let i = 1; i <= 5; i++) {
  console.log("Compte: " + i);
}

Funcions

Una funció és un bloc de codi reutilitzable que fa una tasca concreta. Pot rebre paràmetres i tornar un valor.

Funció amb nom: es declara amb function.

Paràmetres: valors d'entrada.

Return: valor de sortida.

Arrow function: sintaxi moderna =>.

function suma(a, b) {
  return a + b;
}
console.log(suma(3, 4)); // 7

Arrays

Un array és una estructura de dades que emmagatzema múltiples valors en una sola variable, accessibles per índex.

Índex: posició de cada element (comença a 0).

length: nombre total d'elements.

Mètodes: push(), pop(), map(), filter().

let colors = ["vermell", "verd", "blau"];
console.log(colors[0]); // "vermell"
colors.push("groc");
console.log(colors.length); // 4

Programació Orientada a Objectes (POO)

La POO organitza el codi en objectes que combinen dades (propietats) i comportament (mètodes). Les classes són les plantilles per crear objectes.

Classe: plantilla d'un objecte.

Instància: objecte creat a partir d'una classe.

Propietats: atributs de l'objecte.

Mètodes: funcions de l'objecte.

class Gat {
  constructor(nom) {
    this.nom = nom;
  }
  saluda() {
    return `Soc ${this.nom}!`;
  }
}
let meuGat = new Gat("Bigotes");
console.log(meuGat.saluda());
