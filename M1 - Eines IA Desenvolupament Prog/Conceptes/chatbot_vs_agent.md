# Chatbot vs Agent

## ¿Qué es un chatbot?

Un chatbot es un programa de software diseñado para simular una conversación con personas a través de texto o voz. Los chatbots suelen responder preguntas, ofrecer información y ejecutar tareas sencillas basadas en entradas del usuario.

Características de un chatbot:

- Conversación basada en texto o voz.
- Respuestas predefinidas o generadas con modelos de lenguaje.
- Uso común en atención al cliente, asistentes virtuales y FAQ automatizadas.
- Funciona principalmente como interfaz para resolver dudas o completar acciones simples.

## ¿Qué es un agent?

Un agent (agente) es un software que realiza tareas de forma autónoma, tomando decisiones basadas en objetivos, contexto y reglas. Los agentes pueden interactuar con sistemas externos y ejecutar procesos sin intervención continua del usuario.

Características de un agent:

- Actúa de forma proactiva y autónoma.
- Puede planificar y realizar múltiples pasos para cumplir una meta.
- Integra datos, herramientas y acciones externas.
- Suele adaptarse al contexto y aprender de la interacción.

## Diferencias principales

- Un chatbot se centra en la conversación y en responder preguntas del usuario.
- Un agent se centra en lograr objetivos y ejecutar tareas de forma autónoma.
- El chatbot necesita una entrada del usuario para iniciar la interacción.
- El agent puede iniciar acciones por sí mismo y coordinar procesos.

## Ejemplos

- Chatbot: asistente de atención al cliente en una web.
- Agent: asistente que organiza reuniones, envía recordatorios y gestiona tareas automáticamente.
