# ¿Qué es una API?

Una API (Interfaz de Programación de Aplicaciones) es un conjunto de reglas y protocolos que permiten que diferentes aplicaciones o servicios se comuniquen entre sí. Una API define cómo se deben enviar solicitudes y cómo se reciben respuestas.

Características de una API:

- Proporciona una interfaz estandarizada para acceder a funciones o datos.
- Permite que un programa use capacidades de otro sistema sin conocer su implementación interna.
- Facilita la integración entre servicios, aplicaciones y dispositivos.
- Puede usar diferentes formatos de datos, como JSON o XML.

## Ejemplos de uso

- Una aplicación móvil que obtiene datos de un servidor web.
- Un sitio web que muestra información de redes sociales mediante una API externa.
- Un servicio que permite autenticar usuarios o procesar pagos desde otra aplicación.

## Tipos comunes de API

- API web: comunicación entre aplicaciones a través de HTTP/HTTPS.
- API de librería: funciones y métodos expuestos por una biblioteca de software.
- API de sistema operativo: llamadas que permiten acceder a recursos del sistema.
