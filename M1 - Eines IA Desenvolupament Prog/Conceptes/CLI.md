# ¿Qué es una CLI?

CLI significa "Command Line Interface" o "Interfaz de Línea de Comandos". Es un método para interactuar con un sistema usando texto escrito en una terminal o consola.

Características de una CLI:

- Permite ejecutar comandos mediante texto.
- Suele ser más rápida y potente para tareas avanzadas.
- No tiene interfaz gráfica; todo se hace desde la línea de comandos.
- Es muy utilizada por desarrolladores y administradores de sistemas.

## Ejemplos de uso

- Navegar por el sistema de archivos con comandos como `cd` y `dir`/`ls`.
- Ejecutar herramientas de desarrollo, como `git`, `npm` o `python`.
- Automatizar tareas con scripts y comandos encadenados.

## Ventajas de la CLI

- Eficiencia en tareas repetitivas y avanzadas.
- Mayor control sobre el sistema.
- Fácil de automatizar con scripts.

## Desventajas de la CLI

- Curva de aprendizaje más alta para principiantes.
- Menos intuitiva que una interfaz gráfica.
