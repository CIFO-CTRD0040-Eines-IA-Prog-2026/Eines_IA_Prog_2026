# Git y GitHub

## ¿Qué es Git?

Git es un sistema de control de versiones distribuido que permite gestionar los cambios en el código fuente y otros archivos. Con Git se puede:

- Registrar el historial de un proyecto.
- Volver a versiones anteriores del código.
- Trabajar de forma paralela en diferentes ramas.
- Combinar cambios de varios colaboradores.

Git se ejecuta localmente en el equipo del desarrollador y guarda el historial completo del repositorio en cada copia.

## ¿Qué es GitHub?

GitHub es una plataforma de alojamiento de repositorios basada en Git. Ofrece servicios y herramientas para colaborar en proyectos, como:

- Repositorios remotos.
- Pull requests y revisiones de código.
- Issues y gestión de tareas.
- Acciones de integración continua (CI/CD).

GitHub facilita el trabajo en equipo y la publicación de proyectos, al conectarse con repositorios Git locales y sincronizar cambios entre desarrolladores.
