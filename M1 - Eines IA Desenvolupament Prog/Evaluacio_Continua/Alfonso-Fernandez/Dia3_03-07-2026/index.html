Fundamentos de Programación I

Algoritmos y pseudocódigo

Un algoritmo es una secuencia ordenada de pasos que resuelven un problema. El pseudocódigo es una forma de escribir algoritmos en un lenguaje cercano al natural, sin atarse a la sintaxis de un lenguaje de programación concreto.

# Pseudocódigo: calcular la media de dos números
leer a
leer b
media = (a + b) / 2
escribir media

Variables y tipos

Las variables almacenan datos en la memoria. Cada variable tiene un tipo que determina qué clase de valor puede contener: enteros, decimales, texto o booleanos.

# Variables y tipos en Python
nombre = "Ana"        # str
edad = 20             # int
altura = 1.65         # float
es_estudiante = True  # bool

Operadores

Los operadores permiten hacer operaciones sobre los datos: aritméticas (+, -, *, /), de comparación (==, !=, <, >), lógicas (and, or, not) y de asignación (=).

# Operadores en Python
suma = 5 + 3            # 8
es_mayor = 10 > 4       # True
resultado = (5 > 2) and (3 < 7)  # True

Entrada / salida

La entrada permite recibir datos del usuario con input(). La salida muestra información por pantalla con print(). Son la base de la comunicación entre el programa y el usuario.

# Entrada y salida
nombre = input("¿Cómo te llamas? ")
print(f"¡Hola, {nombre}!")

Condicionales

Las estructuras condicionales (if, elif, else) permiten ejecutar diferentes bloques de código según si una condición es verdadera o falsa.

# Condicional
edad = int(input("¿Cuántos años tienes? "))
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

Bucles

Los bucles (for y while) repiten un bloque de código mientras se cumpla una condición o para cada elemento de una secuencia.

# Bucle for
for i in range(5):
    print(f"Vuelta {i + 1}")

# Bucle while
contador = 0
while contador < 3:
    print("Hola")
    contador += 1

Funciones

Las funciones son bloques de código reutilizables que realizan una tarea concreta. Se definen con def y pueden recibir parámetros y devolver valores.

# Función
def saludar(nombre):
    return f"¡Hola, {nombre}!"

mensaje = saludar("Carlos")
print(mensaje)

Fundamentos de Programación II

Listas y colecciones

Las listas almacenan múltiples valores en una sola variable. Pueden contener elementos de diferentes tipos y se pueden modificar añadiendo, quitando o cambiando elementos.

# Listas
numeros = [1, 2, 3, 4, 5]
numeros.append(6)
print(numeros[0])   # 1
print(len(numeros)) # 6

Cadenas (strings)

Las cadenas son secuencias de caracteres. Python ofrece muchas operaciones para trabajar con texto: concatenar, dividir, buscar, reemplazar y mucho más.

# Cadenas
texto = "Programación"
print(texto.upper())       # PROGRAMACIÓN
print(texto[0:5])          # Progr
print(len(texto))          # 12

Recursividad

Una función recursiva es aquella que se llama a sí misma para resolver un problema dividiéndolo en subproblemas más pequeños. Se necesita un caso base para evitar repeticiones infinitas.

# Factorial recursivo
def factorial(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))  # 120

Programación orientada a objetos

La POO organiza el código en clases y objetos. Una clase define un modelo con atributos (datos) y métodos (funciones), y los objetos son instancias de esa clase.

# Clase y objeto
class Perro:
    def __init__(self, nombre):
        self.nombre = nombre
    def ladrar(self):
        return f"{self.nombre} dice guau!"

perro1 = Perro("Rex")
print(perro1.ladrar())

Ficheros

Trabajar con ficheros permite guardar datos de manera persistente. Se pueden abrir en modo lectura (r), escritura (w) o añadir (a) usando open().

# Leer un fichero
with open("datos.txt", "r") as f:
    contenido = f.read()
    print(contenido)

# Escribir un fichero
with open("salida.txt", "w") as f:
    f.write("¡Hola, mundo!")

Manejo de errores

El manejo de errores con try / except permite controlar excepciones en tiempo de ejecución y evitar que el programa se rompa cuando algo sale mal.

# Manejo de errores
try:
    a = int(input("Introduce un número: "))
    b = int(input("Introduce otro: "))
    print(f"División: {a / b}")
except ZeroDivisionError:
    print("No se puede dividir entre cero")
except ValueError:
    print("Debes introducir un número válido")
